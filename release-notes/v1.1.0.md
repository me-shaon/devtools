# DevToolsApp v1.1.0

## Highlights

- Added in-app FAQs across more tools so behavior and limitations are easier to understand without leaving the app.
- Upgraded Text Compare with line diff, git-style unified diff, and character-level detail.
- Added version visibility, update status, and a new in-app "What's New" experience.

## Tool Improvements

- Cron Calculator now uses real cron parsing for the upcoming run list and clearer builder guidance.
- JWT Decoder now labels expiration honestly and makes it clear that signatures are not verified locally.
- Regex Generator and UUID Generator now explain their current behavior and limitations directly in the UI.
- Text Compare now offers richer diff views and more readable output for modified content.

## App Updates

- The sidebar now shows the current app version.
- Available updates can now be checked, downloaded, and installed from inside the app.
- A first-launch "What's New" modal now appears after version bumps and remains available from the sidebar later.
