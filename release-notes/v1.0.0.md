# DevToolsApp v1.0.0

## Highlights

- Shipped the first stable desktop release of DevToolsApp with a cleaner app shell and categorized tool navigation.
- Expanded the toolset with utilities for text, encoding, dates, UUIDs, and unit conversion.
- Improved desktop packaging and CI so the app could ship as a signed, installable desktop build.

## Tool Improvements

- Added Unit Converter with support for length, weight, temperature, and data units.
- Expanded the UUID Generator with more UUID versions and ULID support.
- Improved the URL Encoder, Regex Generator, Text Compare, API Formatter, JSON Viewer, and Date Difference tools with better UI and behavior.

## App Updates

- Reworked the overall desktop UI with a more polished layout and clearer structure.
- Added categorized menu navigation and search to make tools easier to discover.
- Strengthened packaging, icons, screenshots, and build automation for the first stable release.
